== Description ==

WooCommerce Simple Buy Now helps you to add to cart and checkout only one step in the single product page.
You only need to create one button `Buy Now` in `WooCommerce > Settings > WC Simple Buy Now` at before / after or replace Add to cart button. A pop-up will be displayed with the content of the checkout page.

== Installation ==

* From your WordPress dashboard, Visit `Plugins > Add New`
* Search for `woocommerce simple buy`. Click 'Install'.

Or:
* Upload `plugin-name.php` to the `/wp-content/plugins/` directory

* Activate WooCommerce Simple Buy Now from your Plugins page.
* Visit `WooCommerce > Settings > WC Simple Buy Now` and enable it.
* Enjoy!
